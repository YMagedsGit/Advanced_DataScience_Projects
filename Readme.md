# Advanced Data Science with Pandas: Master Portfolio

This repository contains the first eight core projects from the **Python Data Science with Pandas** course. The portfolio demonstrates an end-to-end data engineering and analysis workflow—ranging from complex web API data extraction to high-performance feature engineering.

## Core Competencies
* **Data Engineering:** Handling nested JSON, SQL database integration, and merging multi-file datasets.
* **Data Wrangling:** Cleaning "messy" real-world data, flattening stringified columns, and Unicode normalization.
* **Analytics & Viz:** Explanatory Data Analysis (EDA) using Matplotlib and Seaborn.
* **Machine Learning Prep:** Feature scaling, dummy variables, and sampling strategies.
* **Machine Learning Modeling:** Choosing a model then fine tuning it
---

## 📂 Project Index (1–8)

### Phase 1: The Movies Dataset (Projects 1–5)
*These projects focus on building a data pipeline for a complex movie metadata collection.*
* **01. EDA & Presentation:** Initial deep-dive into movie trends and budget vs. revenue analysis.
* **02. APIs & JSON:** Extracting live data via RESTful APIs and parsing multi-level nested JSON structures.
* **03. Tidy Data Cleaning:** Handling missing values, duplicates, and inconsistent formatting in messy datasets.
* **04. Merging & Transforming:** Joining multiple relational tables and calculating advanced metrics (e.g., ROI).
* **05. SQL Integration:** Loading cleaned Pandas DataFrames into SQL databases and performing parallel queries (As a SQL Training).

### Phase 2: Historical Trends (Projects 6–7)
*Focusing on large-scale data merging using the US Baby Names dataset.*
* **06. Multi-File Concatenation:** Efficiently importing and merging hundreds of text/CSV files into a single unified DataFrame.
* **07. Advanced Visualization:** Analyzing 100+ years of naming trends using pivot tables and complex Seaborn plots.

### Phase 3: Machine Learning Readiness (Project 8)
* **08. Preprocessing, Feature Engineering and Modeling:** Preparing data for predictive modeling. Includes handling categorical variables, standardization, and creating training/test splits from scratch, Then creating a model used Linear Regression as a base model and then used Random Forest Regressor and fine tuned the model .
